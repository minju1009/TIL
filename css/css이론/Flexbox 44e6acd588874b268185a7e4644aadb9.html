😝 Flexbox Froggy 👉 게임하러 가기

  • <개념 1> : container & item
  • <개념2> main axis & cross axis

① display (일반적인 박스들을 플렉스박스로 지정-inline의 속성을 가지게 된다): flex

② flex-direction (축의 방향 지정) : row / row-reverse / column / column-reverse

③ flex-wrap (화면 크기에 따른 배치 지정) : nowrap(기본값) / wrap(화면이 줄어들면 자동적으로 그에 맞춰 줄바꿈) / wrap-reverse

④ flex-flow (flex-direction&flex-wrap 동시 지정) : row nowrap

<!—아이템 어떻게 배치할 것인지 지정—>

⑤ justify-content (main 축의 아이템 배열 지정) : flex-start / flex-end / center / space-around / space-evenly / space-between

⑥ align-items (반대축의 아이템 배열 지정) : center / baseline

⑦ align-content (justify-content 처럼 아이템 배열 지정)

① order(박스들의 순서 지정) : 1 / 2 / 3

② flex-grow : 화면 늘어날 때 박스 비율 지정(아이템 1! 너는 두 배로 크게 채워)

③ flex shrink : 화면 작아질 때 박스 비율 지정 (아이템 1! 너는 두 배로 작게 줄어들어!)

item1{

flex-shrink:2 ;

}

item1{

flex-shrink:1 ;

}

item1{

flex-shrink:1;

④ flex-basis : 화면이 늘어나든, 줄어들든 항상 그 비율 값을 차지해!

⑤ align-self : 아이템 별로 아이템 하나만 정렬이 가능함.
